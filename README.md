# ruined

A topdown 2D online multiplayer game written in [Rust](https://www.rust-lang.org/) with [Bevy Engine](https://bevyengine.org/) set in a ruined city.

## compiling

### initial setup

1. install [rust](https://www.rust-lang.org)
2. `cargo build --release`
